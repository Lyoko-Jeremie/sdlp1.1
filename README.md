#欢迎
你现在看到的是由零之城堡游戏工作室([www.zerocastle.com](http://www.zerocastle.com))负责维护的开源引擎，零号引擎(ZeroEngine)。

##介绍
ZeroEngine是使用c++编写的，基于SDL2的游戏引擎。<br/>
目前为止，引用的第三方库包括：<br/>
SDL2 <br/>
SDL_image <br/>
SDL_mixer <br/>
SDL_ttf <br/>
tinyxml2 <br/>
boost <br/>
